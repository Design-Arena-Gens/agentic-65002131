# Sora Viral Blueprint

Generate two brand-consistent 30s TikTok storyboards (built as 15s hook & payoff clips) tailored around your product and the latest UGC trend archetypes.

## Local Development

```bash
npm install
npm run dev
```

The site runs at `http://localhost:3000`.

## Production Build

```bash
npm run lint
npm run build
```

## Key Features

- Guided brief that captures brand, product, offer, and tone.
- Curated trend archetypes with motion and pacing hints.
- Instant generation of two synced 30s video outlines, each split into two 15s beats.
- Shot, script, CTA, soundtrack, and consistency cues engineered for TikTok + Sora workflows.
